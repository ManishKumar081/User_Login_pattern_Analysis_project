# User_Login_pattern_Analysis_project

## Frontend Setup Instructions

### Tech Stack
- React.js
- React Router DOM
- CSS

### Run Frontend Application

Move to frontend folder

```bash
cd frontend
```

Install dependencies

```bash
npm install
```

Run application

```bash
npm start
```

Runs locally on:

http://localhost:3000

### Current Implemented Features
- Login Page
- Dashboard Page
- Backend API Integration
