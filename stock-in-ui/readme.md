Stock In Page
Description

This project contains a Stock In page developed using React and Material UI. The page is designed to record incoming stock/material entries. It includes an Entry Details section for entering stock information and a Material Details section for managing material records.

The page currently focuses on frontend UI development and form design.



Features
Entry Details form
Entry Type
Entry Date
Posting Date
Warehouse
Supplier
Challan / Invoice Number
Vehicle Number
GRN Number
Reference Number
Notes
Material Details section
Search Material field
Add Row button
Material table structure
Total Quantity and Amount section
Reset, Save as Draft and Submit buttons


Technologies Used
React
Vite
Material UI (MUI)
JavaScript


Prerequisites

Before running the project, make sure the following are installed:

Node.js
npm
Installation

Install all required dependencies:

npm install

Install Material UI:

npm install @mui/material @emotion/react @emotion/styled


Install Material UI Icons:

npm install @mui/icons-material
Run the Application



Start the development server:

npm run dev

Open the application in the browser:

http://localhost:5173
Environment Variables

No environment variables are required for this project.

Database Setup

No database is used in the current version.
